# dataset_colab

Repositório destinado à análise de datasets utilizando o Google Colab.

## Objetivo
Este projeto tem como objetivo centralizar notebooks e materiais para:
- Carregamento e exploração de datasets
- Limpeza e tratamento de dados
- Análises exploratórias (EDA)
- Visualizações e insights

## Como usar
1. Abra os notebooks disponíveis no repositório.
2. Execute-os no **Google Colab** (recomendado) ou em um ambiente Jupyter local.
3. Ajuste os caminhos/links dos datasets conforme necessário.
